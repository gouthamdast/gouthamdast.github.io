# Goutham's Portfolio

A minimalist portfolio website inspired by [lovefrom.com](https://www.lovefrom.com/).

## Features

- Minimalist design with centered wordmark
- SF (San Francisco) font family
- Smooth fade-in animations
- Responsive design for all devices
- Orientation-aware layout adjustments
- Accessibility support (reduced motion)
- Info button for future navigation

## Tech Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript (optional)

## Local Development

Simply open `index.html` in a web browser to view the site locally.
